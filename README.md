# Análise de Cancelamento de Clientes

Projeto de análise de dados com Python para identificar os principais fatores associados ao cancelamento de clientes e propor ações para reduzir a taxa de churn.

## Objetivo
Analisar os dados de clientes, identificar padrões de cancelamento e sugerir soluções viáveis para reduzir o número de cancelamentos.

## Tecnologias Utilizadas
- Python
- Pandas
- Plotly

## Etapas do Projeto
1. Entendimento do problema de negócio  
2. Importação e visualização da base de dados  
3. Tratamento da base de dados  
4. Análise da taxa de cancelamento  
5. Análise das causas do cancelamento e propostas de solução  

## Principais Conclusões
- Clientes com contrato mensal apresentam maior taxa de cancelamento  
- Clientes com mais de 4 ligações ao call center têm maior probabilidade de cancelar  
- Atrasos superiores a 20 dias levam ao cancelamento em praticamente todos os casos  

## Propostas de Solução
- Incentivar a migração de contratos mensais para planos trimestrais ou anuais  
- Criar ações preventivas para clientes com muitas ligações ao call center  
- Intervir antes que o atraso no pagamento ultrapasse 20 dias  

## Como Executar o Projeto
1. Clone o repositório  
2. Abra o arquivo `main.ipynb`  
3. Execute as células no Jupyter Notebook ou VS Code
