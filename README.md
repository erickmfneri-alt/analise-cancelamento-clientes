# Análise de Cancelamento de Clientes

Projeto de análise de dados com foco em identificar os principais fatores
associados ao cancelamento de clientes e simular ações para redução do churn.

## Objetivo
Analisar os dados de clientes, identificar padrões de cancelamento e propor
ações viáveis para reduzir a taxa de cancelamento.

## Tecnologias
- Python
- Pandas
- Plotly

## Resultado
A análise indicou que atuar sobre contrato mensal, excesso de ligações ao
call center e atrasos no pagamento pode reduzir a taxa de cancelamento
de aproximadamente 56% para cerca de 18%.
